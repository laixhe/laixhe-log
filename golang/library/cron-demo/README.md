### cron v3 定时任务

> 安装

```
go get github.com/robfig/cron/v3
```
> 导入
```
import "github.com/robfig/cron/v3"
```

#### 文件格式說明
#### ┌──分鐘（0 - 59）
#### │  ┌──小時（0 - 23）
#### │  │  ┌──日（1 - 31）
#### │  │  │  ┌─月（1 - 12）
#### │  │  │  │  ┌─星期（0 - 6，表示从周日到周六）
#### │  │  │  │  │
#### *  *   *  *  * 被執行的命令
